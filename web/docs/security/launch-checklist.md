# Security Launch Checklist (Web)

- [ ] Threat model reviewed and approved.
- [ ] Auth architecture ADR reviewed and accepted.
- [ ] Security headers and CSP enforced in production.
- [ ] Dependency scan/SBOM generated and policy thresholds satisfied.
- [ ] Security test matrix passed.
- [ ] Incident response and on-call playbooks validated.
- [ ] Data retention/storage policy documented and implemented.
- [ ] Rollback criteria and release owner sign-off recorded.
